# Sistema de Biblioteca em Python

Projeto desenvolvido em Python para praticar lógica de programação, estruturas condicionais, laços de repetição, listas, dicionários e manipulação de dados.

## Funcionalidades

- Cadastro de livros
- Listagem de livros cadastrados
- Busca de livros por título
- Empréstimo de livros
- Devolução de livros
- Remoção de livros
- Validação de entradas
- Tratamento de opções inválidas

## Tecnologias Utilizadas

- Python 3

## Estrutura dos Dados

Cada livro é armazenado em um dicionário contendo:

- Título
- Autor
- Status de empréstimo

Exemplo:

```python
{
    "titulo": "Dom Casmurro",
    "autor": "Machado de Assis",
    "emprestado": False
}
```

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/atila54321/biblioteca-python.git
```

2. Entre na pasta do projeto:

```bash
cd biblioteca-python
```

3. Execute o programa:

```bash
python main.py
```
